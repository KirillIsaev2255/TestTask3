В целом проект реализован достаточно просто. Не успел сделать UI, если надо и дадите больше времени сделаю.
Реализованно все остальное два поля красное и синие, по ним летают дроны(реализованно с помощью NavMesh)
Спавнятся ресурсы дроны их подбирают и несут на базу, процесс повторяется.
Когда ресурс спавнится вызывается Func взависимости от его цвета и база начинает искать ближайшего к нему свободного дрона,
после этого дрон летит забирать ресурс и возвращает на базу, после чего он становится свободным и срабатывает эффект, реализованный с помощью Particle System
